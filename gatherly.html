Gatherly

In Development

A Discord-like real-time communication app featuring servers/channels, messaging, voice rooms, screen sharing, and presence.

React Node.js Socket.io WebRTC SimplePeer MongoDB Atlas

Key Features
  • Server & channel system with role/permission management
  • Real-time messaging (Socket.io) with typing indicators
  • Voice rooms & screen sharing (WebRTC + SimplePeer)
  • Presence/online status and basic moderation tools
  • File sharing and message attachments
  • REST API documented with Swagger
Roadmap
  1. Messaging & channels (completed – MVP)
  2. Voice rooms & screen sharing (prototype)
  3. Role/permission model & moderation
  4. Notifications & offline message sync
  5. Production hardening: Docker, CI/CD, observability
